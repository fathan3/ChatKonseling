# Online Counseling Chat - Mediator Pattern
Implementasi Design Pattern pada aplikasi Chatting.

## Fitur Utama
- **Mediator Pattern:** Pusat kendali pesan untuk mengurangi dependensi antar objek.
- **Modern UI:** Menggunakan Tailwind CSS untuk tampilan chat yang responsif.
- **Fast Backend:** Dibangun dengan FastAPI untuk performa tinggi.

## Tech Stack
- Python (FastAPI)
- Tailwind CSS
- HTTP Endpoints
